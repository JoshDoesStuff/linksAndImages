Character Sets

You can use UTF-8 to count in Turkish:

  1. bir
  2. iki
  3. üç
  4. dört
  5. beş

HTML Entities

There are 3 reserved characters in HTML: < > and &. You should always use HTML entities for these 3 characters.

If you’re into “web typography,” you’ll also find yourself using curly quotes quite a bit.
